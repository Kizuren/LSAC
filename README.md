# LSAC
Limited Service ACcess

## Intro
LSAC is a small program inspired by a friend of mine who made a script which gives the user only some options, instead of allowing the user to use the bash like normally.

## How to use it
It's really simple!

You run the setup to generate the config:
```
LSAC setup
```

Afterwards you just set the ssh command to be:
```
LSAC
```

It's that easy!
